pre




turorial de como usar a funçao pre do html,
O texto em um <pre> elemento é exibido em uma fonte de largura fixa
e preserva os espaços e as quebras de linha. O texto será exibido
exatamente como escrito no código-fonte HTML.

img

turorial de como usar a funçao img do html.
A <img>tag é usada para inserir uma imagem em uma página HTML.

As imagens não são tecnicamente inseridas em uma página da web; as imagens estão vinculadas a páginas da web. A <img>tag cria um espaço de retenção para a imagem referenciada.

Nota: Além disso, sempre especifique a largura e a altura de uma imagem. Se a largura e a altura não forem especificadas, a página pode piscar enquanto a imagem é carregada.
Dica: Para vincular uma imagem a outro documento, simplesmente aninhe a <img>tag dentro de uma tag (veja o exemplo abaixo).

video

turorial de como usar a funçao video do html. O controlsatributo adiciona controles de vídeo, como reprodução, pausa e volume.

É uma boa ideia sempre incluir widthe heightatributos. Se a altura e a largura não forem definidas, a página pode piscar enquanto o vídeo carrega.

O elemento permite que você especifique arquivos de vídeo alternativos que o navegador pode escolher. O navegador usará o primeiro formato reconhecido.

O texto entre as tags <video>e <video>só será exibido em navegadores que não suportam o <video>elemento.
